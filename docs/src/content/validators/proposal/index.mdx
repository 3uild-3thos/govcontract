# Proposal

Display a specific governance proposal and its details.

## Description

This command retrieves and displays detailed information about a specific governance proposal, including its status, votes, and metadata.

## Arguments

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `<PROPOSAL_ID>` | String | Yes | The proposal's ID (PDA, base58) to display |

## Global Arguments

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `--rpc-url`, `-r` | String | No | `SVMGOV_RPC` env var | Custom RPC URL |

## Example

```bash
svmgov proposal "ABC123..." \
  --rpc-url https://api.mainnet-beta.solana.com
```

## Output

The command displays:

- Proposal title and description
- Author and creation timestamp
- Status (support period / voting / finalized)
- Current vote totals (For, Against, Abstain)
- Cluster support
- Snapshot slot and consensus result (if present)
